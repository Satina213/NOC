What is Randomness

When we talk about something random we mean to say that whatever the result is it will be from some range of possibilities.
The first way we can think about this is with an even distribution. A roll of the dice has an even distribution because each of the faces has an equal chance of showing up. Or we can draw 1000 pixels on a canvas and give each one a random x and random y position so we can see the randomness.
random()

We can also have what's known as a Gaussian Distribution. Think of this like people's height where any person is a random height but you have more people of average height and fewer and fewer people exist that are heights further from the average.
The standard deviation is calculated as the square root of the average of the squares of deviations around the mean. -Nature of Code, pg 15
In other words we find the average value of the data, find the difference each value is from the average, square each difference, find the average squared difference, and take the square root of that.
In other other words the higher the standard deviation the wider the range of possibilities. A low standard deviation will have everything clumped around the mean. randomGaussian()

A random distribution of points with a high standard deviation.
A random distribution of points with a low standard deviation.

So we've covered an even distribution with random() and a Gaussian distribution with randomGaussian() and for the final entry into randomness we have Perlin noise().
Given any input noise will return a random value between 0 and 1 but through complicated mathematics tiny changes to the input will result in random but tiny changes to the output. so noise(0) might return 0.56 and noise(0.01) would return 0.57 and so on. You'll never know what's coming next but it's always going to be close to whatever the previous output was.
