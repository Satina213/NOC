What is a vector? A vector has both magnitude and direction and is commonly denoted by an arrow. The size of its arrow is its magnitude and its heading is its direction.

Here we're using vectors to draw a line from the origin (top left) to the mouse.
We can draw a vector from the middle of the screen to the mouse by taking the mouse
vector and subtracting the vector to the origin of the screen.


I'm using colors to demonstrate the addition and subtraction going on.
The Blue and Yellow vectors add together to make the Green vector.
The Blue vector is the Green minus the Yellow and the Yellow is the Green minus the blue.

You can think of adding vectors like taking the tail of the second and sticking it to the head of the first. Wherever you end up is the result of addition.

For subtraction (vector A minus vector B) we flip vector B 180 degrees and then add A to the reversed B.

So how are we using vectors to make life any easier? With vectors we keep track of one entity unlike in 2D when we need an x and y, or in 3D when we need a z as well. Let's take a look at some of the code for the following "sketch". 

    function setup() {
        createCanvas(640, 240);
        background(100);
        acceleration = createVector(0, 0);
        velocity = createVector(0, 0);
        position = createVector(width / 2, height / 2);
        mouse = createVector(mouseX, mouseY);
    }
    function draw() {
        background(100);
        circle(position.x, position.y, diameter);
        mouse.x = mouseX;
        mouse.y = mouseY;

        acceleration.set(mouse.sub(position));
        acceleration.setMag(1);
        acceleration.div(mouse.sub(position));
        velocity.add(acceleration);
        velocity.limit(10);
        position.add(velocity);
        if (position.x > width - diameter / 2) {
            velocity.x *= -1;
            position.x = width - diameter / 2;
        } else if (position.x < 0 + diameter / 2) {
            velocity.x *= -1;
            position.x = diameter / 2;
        } else if (position.y > height - diameter / 2) {
            velocity.y *= -1;
            position.y = height - diameter / 2;
        } else if (position.y < 0 + diameter / 2) {
            velocity.y *= -1;
            position.y = diameter / 2;
        }


    }
