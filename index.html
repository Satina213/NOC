Nature of Code

What I'm doing

Here I'm following along with Daniel Shiffman's 'Nature of Code' book and incorporating it into this website. I hope that I will learn some web design as well as javascript with this project. More specifically I want to learn concepts that can be applied to future projects, specifically those to do with simulating reality or in other words: 'Nature of Code'!

The layout of this website

As of right now I'm breaking this up into three large sections: HTML, CSS, Javascript (specifically p5.js). The Javascript section of this website will be subdivided into chapters following the book and will be the most extensive.
